# 📊 E-Commerce Sales & Profit Optimization Analysis

## 🎯 Project Overview
This project aims to analyze an e-commerce store's data to understand why profits are declining despite stable sales performance. 

## 🛠️ Tech Stack & Learning Roadmap
I am building this project in phases as I master the data analytics tool stack:
- [x] **Phase 1: Data Cleaning & Baseline Insights** — Excel (Completed)
- [ ] **Phase 2: Advanced Querying & Database Management** — SQL (In Progress)
- [ ] **Phase 3: Interactive Dashboarding** — Power BI (Upcoming)
- [ ] **Phase 4: Predictive Analysis** — Python (Upcoming)

## 📁 Repository Structure
- `/Data/` : Contains the raw and cleaned Excel datasets.
- `/SQL_Queries/` : Will contain scripts for deep-dive analysis.
- `/Dashboards/` : Will hold the Power BI (.pbix) files and screenshots.
- `/Python_Notebooks/` : Will contain Jupyter notebooks for automated cleaning.
